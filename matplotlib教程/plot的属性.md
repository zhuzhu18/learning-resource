matplotlib.pyplot.plot(*args, \***kwargs)的参数
<pre>
plot(x, y)        # plot x和y使用默认线条样式和颜色，即蓝色直线
plot(x, y, 'ro)   # plot x和y使用红色圆圈标记
plot(x, y, 'r+')  # 使用红色加号标记
plot(x1, y1, 'g^', x2, y2, 'g-')   # 绘制一条绿色线条和一组绿色标记
'-'	实线
'--'	长虚线（区别于点虚线）
'-.'	点划线
':'	点虚线
'.'	点标记
'o'	圆圈标记
'v'	下三角标记
'^'	上三角标记
'<'	左三角标记
'>'	右三角标记
